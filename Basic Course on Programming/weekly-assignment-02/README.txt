a) What is the difference between a class and on object?
- Luokka on tietotyyppi kun taas olio on, jonka tietotyyppi on jokin luokka.


b) What is public interface?
- Toiminta joka näkyy ohjelmassa/luokassa ulospäin, eli muuttuja/funktiot yms tiedetään etukäteen.


c) What is private interface?
- Toiminta johon ei näe muuta kuin ohjelmoija itse. Ulospäin ei tarvitse tietää miten joku toiminta on tehty kunhan antaa tiedot julkisen rajapinnan mukaan. Yksityisen rajapinta voidaan siis toteuttaa monella tavalla eikä sen toiminnasta tarvitse tietää tarkemmin.
